# Timeline Game

A simple New York Times-style layout for a timeline quiz game.

## Project structure

- `index.html` — home page for the game experience
- `game.html` — game page with the timeline challenge
- `results.html` — results page placeholder
- `base.html` — generic site layout shell
- `css/styles.css` — visual styling
- `js/app.js` — basic game behavior and drag-and-drop logic
- `images/` — folder for artwork and event graphics
- `claude.md` — design notes and instructions

## Getting started

1. Open `index.html` in a browser.
2. Click **Play the Game** to go to `game.html`.
3. Click **Start Game** to reveal the timeline challenge.
4. Drag event cards into the timeline slots.
5. Click **Check Order** to see round feedback.
