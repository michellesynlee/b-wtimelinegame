# Timeline Game Notes

This game is designed to mimic a NYT-style feature layout with a timeline twist.

## Purpose

- Use a clean, editorial page layout.
- Let players sort historical events into the correct order.
- Keep the interface simple and mobile-friendly.

## Structure

- `index.html`: main interactive page.
- `base.html`: reusable page shell for future layouts.
- `css/styles.css`: typography, spacing, timeline board, card styling.
- `js/app.js`: game state, round flow, drag and drop logic.
